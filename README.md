## About Account Settings on GitHub

GitHub provides separate settings for each account, meaning you can manage settings independently for personal and organizational accounts.

## Managing Account Settings

You need to manage settings and features for each of your accounts separately.

## Switching Between Account Settings

If you're an organization or enterprise owner, you can switch between settings for different accounts using the context switcher in your settings.

1. At the top of the page, to the right of your name, click **Switch settings context**.
2. Start typing the name of the account you want to switch to, then click the name of the account.
3. In the left sidebar, click **Settings**.
